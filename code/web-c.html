บทคัดย่อ

โปรเจคนี้เป็นระบบที่ใช้ Microcontroller เพื่อควบคุมประตูและหลอดLED โดยใช้ Ultra sonic sensor และ LDR sensor โดยโปรเจคนี้จัดทำเพื่อการเปิดขปิดอัตโนมัติสำหรับประตูและระบบแสงในที่อยู่อาศัยหรือสถานที่ทำงาน โดยมีความสามารถที่จะวัดระยะห่างของสภาพแวดล้อมด้วย Ultra sonic sensor และวัดค่าความสว่างด้วย LDR sensor อย่างแม่นยำ โปรแกรมใน Microcontroller ถูกออกแบบมาเพื่อทำให้ประตูเปิดหรือปิดตามระยะทางที่ตั้งไว้ และควบคุมการเปิด-ปิดของไฟในตอนกลางคืนโดยอัตโนมัติ โดยใช้ข้อมูลที่ได้จาก LDR sensor

ฟังก์ชันหลัก

1. Ultra Sonic Sensor: ใช้สำหรับวัดระยะห่างของวัตถุที่เข้ามาใกล้ประตูและเปิดหรือปิดประตูอัตโนมัติโดยใช้ motor servo

2. LDR Sensor: ใช้สำหรับวัดระดับแสงและควบคุมการเปิด-ปิดของไฟในตอนกลางคืน

หลักการทำงาน

ระบบสามารถทำงานอัตโนมัติโดยไม่ต้องการการควบคุมจากผู้ใช้ เมื่อตรวจจับว่ามีคนเข้าใกล้ประตูหรือในสภาพแวดล้อมที่มืดลงการปรับแต่งค่าได้ : ผู้ใช้สามารถปรับค่าต่างๆ เช่น ระยะทางของการเปิดปิดประตู หรือระดับความสว่างที่ต้องการในการเปิด-ปิดไฟ
